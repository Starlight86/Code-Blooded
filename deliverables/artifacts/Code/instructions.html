Instructions / Tutorial Page

Welcome to the Instructions page for the Riki website. Here, you'll find detailed explanations on how to utilize various features of our platform.

Feature 1: Switch the page to a dark mode template

Description: Dark mode is a button in the navbar that when hit will cycle the page to a dark theme and when hit again cycle the page back to the light theme.

Feature 2: Allow users to set a custom template for a page

Description: Custom template will let users set the background color and text color for pages. Will likely use the button from feature 1 and cycle though light, mode dark mode and custom page theme

Feature 3: Text visibility modifier for visually impaired (High Contrast / Text Size)

Description: Accessibility Mode has a button on the Navbar of every page that when pressed opens a pop-up window. In this pop-op window are two checkboxes labeled high-contrast mode and larger text. the high contrast checkbox, when selected, changes the background to black and the text color to white where possible. this is for users that have visual impairments that reduce the ability to distinguish similar colors. larger text, when selected, increases all text size, where possible, by 50% for users that struggle to see small print.

Feature 4: Citation Generator

Description: Citation Generator is a button that will be at the bottom of each article/page and when pressed creates a pop-up window containing pre-made citations in MLA and APA format for that article.

Feature 5: Instructions page and Tutorial.

Description: To access the features of our website, navigate through the following options:

  • Home: Click on the "Home" link to go to the homepage of our website.
  • Index: Access the index of pages by clicking on the "Index" link.
  • Tags: Explore pages based on tags by clicking on the "Tags" link.
  • Search: Use the search functionality by clicking on the "Search" link.

Additionally, you can create a new page by clicking on the "New Page" link.

Feature 6: Role-based Access to Admin Page

Description: This feature implements role-based access control to the admin page of our website. Users with specific roles, such as "admin," are granted access to the admin page, where they can perform administrative tasks. By applying roles, we ensure that only authorized users can access sensitive functionalities, maintaining the security and integrity of our website. This feature enhances user management and security by restricting access to privileged areas based on predefined roles.
